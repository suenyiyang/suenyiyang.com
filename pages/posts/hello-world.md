# Hello world, again

## 我又要开始写博客了

我又要开始写博客了。

2019 年，刚上大学的时候，当时同学里流行搭博客。最初版的博客是基于 Typecho 搭建的。Typecho 基于 PHP 语言，当时第一次听说了「LNMP」“架构”。

用了几款主题，想设计一套属于自己的博客主题，所以做了 [`18px-typecho-theme`](https://github.com/suenyiyang/18px-Typecho-Theme)。

后来一段时间没有维护服务器，博客荒废了。

2021 年，疫情在家，用 hexo 搭了自己第二个博客。做了第二个主题 [`hexo-theme-linear`](https://github.com/suenyiyang/hexo-theme-linear)。

Hexo 很强大，但是也很慢。加上需要定制主题和功能的话，需要用 Stylus 和 EJS 写样式和模板，工程化比较麻烦，后来慢慢的也舍弃了。

2022 年，大三的时候，在北京实习。对前端有一星半点的了解，基于 VitePress [搭了一个博客](https://yysuen.me)，部署在 Netlify 上（[备用链接](https://yysuen.netlify.app)）。

同样，也给 VitePress 写了一个博客主题 [`vitepress-theme-linear`](https://github.com/suenyiyang/blog/tree/main/packages/theme)。

这种 serverless 的部署方案比 GitHub Pages 更灵活，而且不用自己维护服务器，部署的内容是静态页面，也不用担心被攻击。

但当时脑子里没什么东西，写了几篇之后，没有持续更新了。

工作之后，承蒙同事信任，在前端一些方向有自己的探索和思考。加上业务主要用 React，想试试 React 19 和 NextJS，所以开始搭现在这个博客。

## 关于博客的计划

在内容上，预计会以工作中和平时遇到的前端问题、解决方案、知识沉淀为主，在不泄露过多个人隐私的前提下，可能也会有一些生活、思考方面的记录。

之前写博客，总会落入一些误区。

比如喜欢写一些有噱头的内容，但是由于没有深入思考，过一段时间再回来看，自己都唾弃之前的内容。

比如担心自己的知识积累不足以支撑博客内容，写了之后没人看，于是懒得写。

这次再开始写博客，主要想写给自己看，在写作中加深自己的思考，锻炼自己的文字表达能力，沉淀学习到的知识。定期回看自己的博客，一来温故知新；二来看看自己在过去的时间里，理解是否加深，思维有没有迭代。

如果发现之前有没想清楚，没记录对的地方，及时修改，以免万一博文被其他人看到，误导别人。

总之，这次重新写博客，希望自己能更专注于内容的积累。

在形式上，主要有 4 个 milestone：

1. 支持基础博客能力；
2. 优化阅读体验；
3. 添加个性化功能；
4. 技术优化；

目前刚刚实现第一个 milestone，然后就迫不及待地写下这篇博客了。

## 独立博客还有存在的必要吗

在自己写下这篇博客当天，看到一位前端大佬的博客挂上了停服的通知。之前一起折腾搭博客的同学，也都放弃了维护个人博客。

越来越多的 App 只允许应用内搜索，传统搜索引擎逐渐被大模型取代，维护个人博客真的还有意义吗？

对我而言，是有意义的。

通过写下来这些文字，自己可以作为一个“赛博史官”，记录技术和生活中宏观、微观的历史；可以和自己、和其他人跨时空地交流。另外，随着 AI 的发展，或许等到自己百年之时，能够把一些文字、影像资料投喂给 AI，自己作为一个“赛博人”获得永生……

---

总而言之，我又要开始写博客了。Hello world, again！
